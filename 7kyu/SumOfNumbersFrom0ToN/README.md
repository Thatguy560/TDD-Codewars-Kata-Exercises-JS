## Sum of numbers from 0 to N Kata

### Requirements 


We want to generate a function that computes the series starting from 0 and ending until the given number.

Example:

Input:

```
<6
```

Output:

"0+1+2+3+4+5+6 = 21"

### Test Cases

```JavaScript
SequenceSum.showSequence(6),"0+1+2+3+4+5+6 = 21"
```

### Input/Output Table

| Input                                          | Output |
| :--------------------------------------------- | :----- |
| 6                           | "0+1+2+3+4+5+6 = 21"      |
| 0                             | "0=0"      |
