## Sum of angles Kata

### Requirements 

Find the total sum of internal angles (in degrees) in an n-sided simple polygon. N will be greater than 2.

### Test Cases

```JavaScript
angle(3), 180)
angle(4), 360)
```

### Input/Output Table

| Input                              | Output |
| :---------------------------------| :----- |
| 3                                  |   180    |
| 4                                  |  360     |
